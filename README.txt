HƯỚNG DẪN THÊM ẢNH

1. Bỏ ảnh của bạn vào thư mục:
assets/images/products/

2. Tên file ảnh cần khớp đúng với file đã khai báo trong:
assets/js/products.js

Ví dụ:
image: "assets/images/products/chatgpt-pro-slot.png"

=> bạn chỉ cần chép ảnh của bạn vào thư mục products và đặt tên:
chatgpt-pro-slot.png

3. Web này đã sửa sẵn:
- không còn hiện tên file ảnh trên giao diện
- ảnh lỗi sẽ tự ẩn
- nếu chưa có ảnh thì hiện icon đẹp thay thế
- danh mục hiển thị đúng

4. Mở web bằng Live Server hoặc mở file index.html trực tiếp.
